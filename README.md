# Personal Expense Tracker

A simple console-based .NET Framework application used to add, view, and manage personal daily expenses.

##  Project Overview
The **Personal Expense Tracker** helps users to:

- Add new expenses  
- View all expenses  
- Filter by category  
- Calculate total and average expenses  
- Store expenses during runtime  


## Technologies Used
- C#  
- .NET Framework (Console Application)  
- Visual Studio  
- Collections (List)  

##  How to Create This Project in Visual Studio

Follow these steps to create the project:

1. Open **Visual Studio**
2. Click **Create a new project**
3. Select **Console App (.NET Framework)**
4. Click **Next**
5. Enter the Project Name: **Personal Expense Tracker**
6. Choose the Framework (example: .NET Framework 4.7.2)
7. Click **Create**
8. Your Program.cs file will open — start writing your code here.
   
##  How to Add `Expense.cs` File in Visual Studio

1. In **Solution Explorer** → Right-click your project name (`PersonalExpenseTracker`)
2. Click **Add**
3. Select **Class**
4. Set the name as **Expense.cs**
5. Click **Add**
6. Replace the code.
